# FamilySync ✨

> ⚠️ **Disclaimer:** This application is focused on privacy, fully *self-hosted*, and designed exclusively for single, private family use.

**FamilySync** is a residential family management portal designed with an ultra-modern **Dark Glassmorphism** aesthetic. It combines a cooperative, smart shopping list, gamified chore routines with isolated privacy, AI-assisted medication scheduling, and a dedicated Fridge Mode for wall displays (*Smart Home Wall Dashboards*).

---

## 🚀 Key Features

*   🛒 **Pantry & Shopping Lists via AI (Gemini)**: Real-time item classification and categorization using the `gemini-2.5-flash` model integrated with a zero-latency (0ms) *Cache-First* flow in local IndexedDB, alongside fluid native *Drag & Drop*.
*   🔒 **Role-Based Access Control (RBAC)**: Physical separation of privileges between Administrator (full management, member administration, password resets, backups, and AI settings) and Regular Members.
*   🛡️ **Chore Isolation & Privacy**: Regular members only view tasks assigned to them or marked for "All Family." Only Administrators have a radar-view of all tasks in the household.
*   💊 **Smart Medication Scheduling**: Assisted cascade scheduling panel. Set the daily dose frequency (e.g., every 8 hours) and treatment duration in days, and the system autonomously schedules all matching tasks.
*   💾 **Backup Hub (Disaster Recovery)**: Secure transactional export and import of family data in JSON format, along with an automatic, rotating local backup system inside IndexedDB (keeping up to 3 recent snapshots).
*   🔔 **SSE Notifications & Audio Chime**: Real-time synchronization across all devices in the home via Server-Sent Events (SSE), complete with operating system push notifications and a playful audio chime programmatically generated by the Web Audio API (0 bytes to download!).
*   📺 **Fridge Mode (Smart Home Wall Dashboard)**: Wall-optimized dashboard for home tablets with a digital clock displaying seconds, time-based seasonal greetings, a quick-add bar integrated with Gemini AI, and premium artistic empty states.
*   🧹 **Closed Registration (Hardening)**: Public registration is suspended (`403 Forbidden`) for maximum security against unauthorized external access. Family members are added manually by the household administrator.

## 🐳 Quick Start (Docker)

The recommended way to run FamilySync in production is through Docker Compose, ensuring full isolation and secure persistence.

### 1. Create the `docker-compose.yml` file

```yaml
version: '3.8'

services:
  familysync:
    image: gutolm/familysync:latest
    container_name: familysync-app
    restart: unless-stopped
    ports:
      - "5000:5000"
    environment:
      - PORT=5000
      - DATABASE_PATH=/data/familysync.db
      - JWT_SECRET=FamilySyncSecretKey_SuperSecure123! # Change to a strong, secure secret key!
    volumes:
      - familysync-data:/data

volumes:
  familysync-data:
    driver: local
```

### 2. Run the container

```bash
docker compose up -d
```

Access the interface in your browser: **`http://YOUR_SERVER_IP:5000`**
*   **First-Time Login**: Username: `admin` / Password: `admin`
*   *We strongly recommend going to Settings > My Profile and updating your username and password immediately after your first login!*

## 🛠️ Manual Installation (Development)

If you would like to run the local development environment directly on your machine:

### Requirements
*   [Node.js](https://nodejs.org/) v20 or higher.
*   [NPM](https://www.npmjs.com/) v10 or higher.

### Initialization Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mwatney-mars/FamilySync.git
    cd FamilySync
    ```

2.  **Configure Environment Variables:**
    Create a `.env` file in the root directory based on `.env.example`:
    ```env
    PORT=5000
    DATABASE_PATH=./backend/familysync.db
    JWT_SECRET=YourUltraSecretKeyHere
    ```

3.  **Install dependencies and start the Backend:**
    ```bash
    cd backend
    npm install
    npm run dev
    ```

4.  **Install dependencies and start the Frontend (Vite):**
    Open another terminal:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
